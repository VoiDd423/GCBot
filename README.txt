Для запуска бота нужно:

1. Скачать на сервер Python :

Ubuntu/Debian: - 
--------------------------------------------------------------------
	sudo apt update
	sudo apt install python3
--------------------------------------------------------------------

CentOS/RHEL:
--------------------------------------------------------------------
	sudo yum install epel-release
	sudo yum install python3
--------------------------------------------------------------------

2. Открыть файл config и заменить TEST_TOKEN и MONGO_URI на ваши данные

3. Открыть в консоли папку проекта

4. Прописать в консоль :
--------------------------------------------------------------------
	pip install discord.py pymongo re dateutil motor PIL io requests 
--------------------------------------------------------------------

5. После загрузки всех библиотек для запуска бота прописать:

--------------------------------------------------------------------
	python3 bot.py
--------------------------------------------------------------------

6. Если все заработало то поздравляем у вас все получилось! Если нет, то значит не ваша вина а разработчик дебил.
